# Password Strength Checker

A simple Python project to:
- Check password strength
- Generate strong passwords
- Save results into a report file

## Files
- `app.py` - Main program
- `checker.py` - Password strength logic
- `generator.py` - Password generator
- `report.py` - Save results to text report
- `report.txt` - Generated report output

## How to run

```bash
python app.py
